# RimWorld 游戏框架 - 图形界面版本

## 概述

RimWorld游戏框架现在包含了一个完整的图形用户界面，提供更直观的用户体验。

## 功能特性

### 主界面
- **现代化设计**: 采用深色主题，符合游戏风格
- **开始游戏按钮**: 点击进入游戏演示界面
- **结束游戏按钮**: 安全退出应用程序
- **状态显示**: 实时显示程序状态和版本信息

### 游戏界面
- **实时演示**: 显示RimWorld框架的实际运行情况
- **控制面板**: 启动/停止演示，清空输出等功能
- **游戏输出**: 实时显示游戏框架的运行日志
- **返回主菜单**: 随时返回主界面

## 启动方式

### 方式一: 图形界面版本（推荐）
```bash
run-demo.bat
```

### 方式二: 控制台版本（备用）
```bash
run-console-demo.bat
```

## 系统要求

- Windows 10 或更高版本
- .NET 8.0 SDK 或运行时
- 支持WPF的环境

## 项目结构

```
src/

│   ├── MainWindow.xaml             # 主窗口界面
```

## 使用说明

1. **启动程序**: 双击 `run-demo.bat` 或在命令行中运行
2. **主界面**: 程序启动后显示主菜单界面
3. **开始游戏**: 点击"🎮 开始游戏"按钮进入游戏界面
4. **游戏演示**: 在游戏界面中点击"启动演示"查看框架运行
5. **返回主菜单**: 点击"返回主菜单"或关闭游戏窗口
6. **退出程序**: 在主界面点击"❌ 结束游戏"

## 技术实现

### 界面技术
- **WPF**: Windows Presentation Foundation
- **XAML**: 声明式界面设计
- **MVVM模式**: 数据绑定和事件处理

### 核心功能
- **进程管理**: 启动和管理演示进程
- **实时输出**: 捕获并显示程序输出
- **事件处理**: 响应用户交互
- **状态管理**: 跟踪程序运行状态

## 故障排除

### 常见问题

1. **程序无法启动**
   - 确保已安装 .NET 8.0 SDK
   - 检查Windows版本是否支持WPF

2. **演示无法运行**
   - 程序会自动回退到控制台版本
   - 检查项目文件是否完整

3. **界面显示异常**
   - 确保显示器分辨率足够（最小800x600）
   - 检查Windows主题设置

### 日志文件
程序运行日志会显示在游戏界面的输出区域中，包含：
- 构建过程信息
- 演示运行状态
- 错误信息和警告
