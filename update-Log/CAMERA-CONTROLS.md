# 🎮 相机控制系统

## 新增功能

为游戏世界界面添加了完整的相机控制系统，支持键盘和鼠标操作。

## 🖥️ 全屏体验

### 全屏模式
- **自动全屏**: 游戏世界窗口自动以全屏模式启动
- **无边框**: 移除窗口边框，提供沉浸式体验
- **退出方式**: ESC键或右上角退出按钮

## 🎯 控制方式

### 键盘控制 (平滑移动)
- **WASD** 或 **方向键** - 移动视角
  - `W` / `↑` - 向上移动视角
  - `S` / `↓` - 向下移动视角  
  - `A` / `←` - 向左移动视角
  - `D` / `→` - 向右移动视角
  - **平滑移动**: 长按键盘无卡顿，60FPS流畅移动

### 特殊功能键
- **空格键** - 快速居中到人物位置
- **R键** - 重置缩放级别到1.0x并居中
- **F键** - 切换自动跟随人物模式
- **ESC键** - 退出全屏模式

### 鼠标控制
- **鼠标滚轮** - 缩放视角
  - 向上滚动 - 放大 (最大5.0x)
  - 向下滚动 - 缩小 (最小0.1x)
  - 无需按住Ctrl键，直接滚动即可

## 📊 技术参数

### 移动参数
- **移动速度**: 3.0像素/帧 (平滑移动)
- **更新频率**: 60FPS (16ms间隔)
- **缩放速度**: 0.1倍/滚轮刻度
- **缩放范围**: 0.1x - 5.0x
- **默认缩放**: 1.0x

### 平滑移动系统
- **无卡顿**: 基于定时器的连续移动
- **多键支持**: 支持同时按下多个方向键
- **即时响应**: 按键按下立即开始移动
- **精确控制**: 松开按键立即停止移动

### 智能功能
- **自动跟随**: 开启后相机会自动跟随人物移动
- **手动控制优先**: 使用WASD移动时自动关闭跟随模式
- **实时信息**: 状态栏显示当前缩放级别、视角位置和跟随状态
